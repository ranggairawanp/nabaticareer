Admin Module
Dulu semua konfigurasi TA tersebar di banyak layar di bawah satu Master Admin. Konsol ini menyatukannya jadi satu pintu, lalu menandai kepemilikan tiap domain sesuai dekomposisi tata kelola. Admin Module mengatur yang operasional, sementara kunci sensitif tetap milik Platform Admin, Compliance dan Governance, atau DPO.

Catatan: tautan tiap kartu membuka editor detail domainnya. Perubahan disimpan di editor masing-masing, dan kunci sensitif memerlukan persetujuan pemilik tata kelola dengan maker-checker.
