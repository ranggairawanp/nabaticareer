# Nabati Talent Acquisition, Demo Publik

Peragaan modul Talent Acquisition One HCMS untuk Nabati Group. Versi ini disiapkan untuk hosting publik seperti GitHub Pages, jadi tanpa gerbang login.

## Rute

- Akar membuka landing induk.
- career membuka situs karier publik untuk kandidat, sepuluh bahasa.
- console membuka konsol rekrutmen internal dengan sepuluh persona, langsung ke pemilih persona tanpa login.

## Catatan

- Ini lingkungan demo. Seluruh data bersifat contoh dan tersimpan sementara di peramban pengunjung.
- Seluruh tautan relatif, aman digelar di subpath maupun akar domain.
- Desain mengikuti One HCMS Design System v3, Schibsted Grotesk, Geist, Geist Mono, warna merek merah Nabati.
- Versi ini sengaja tanpa otentikasi karena GitHub Pages bersifat publik. Bila konsol harus benar-benar tertutup, gelar ke host dengan proteksi kata sandi seperti Vercel atau Netlify, dan gunakan paket bergerbang penuh.
