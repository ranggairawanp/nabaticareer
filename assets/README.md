# Assets

## brand/nabati-logo.png
The Nabati mark, a red droplet with the white wordmark, on transparent
background. There is no green leaf. Brand red is #E1231B.

## images/
Source photography used across the candidate site, compressed and embedded into
`index.html` at build time. Keep these as the editable source. To swap a photo
on the site, replace the matching embedded image in `index.html`.

## Fonts
Bricolage Grotesque, Geist, and Geist Mono are embedded directly inside both
HTML files as web fonts, so the brand typography renders with no external
dependency. Plus Jakarta Sans is not used anywhere and must not be introduced.
